# Hydrion Connected Devices Roadmap

Status: roadmap-only for v1 owner review.

Hydrion's active v1 runtime does not scan for Bluetooth devices, connect to a
smart bottle, request HealthKit/Google Fit/Health Connect permissions, or read
watch/wearable data.

The visible Connected Devices Coming Soon surface covers future:

- BLE smart bottle sync;
- smartwatch or wearable hydration-context sync;
- clear permission, privacy, and source-attribution copy before any native
  adapter is enabled.

Until a real adapter, permission flow, data model, owner-approved privacy copy,
and device validation exist, connected devices must remain inactive and must not
emit fake hydration data.
